# 🏆 Polla Mundialista 2026

Aplicación web desarrollada en **HTML5** y **CSS3** para visualizar la clasificación de los participantes de la **Polla Mundialista 2026**.

La página muestra dos tablas de posiciones:

* Canal Digital
* Canal Telefónico

Cada participante está asociado a una selección nacional mediante su respectiva bandera y se muestran los puntos obtenidos en cada jornada junto con el total acumulado.

# 🌐 Publicar en GitHub Pages

1. Suba el proyecto a GitHub.

2. Ingrese al repositorio.

3. Abra:

```
Settings
```

4. Seleccione:

```
Pages
```

5. En **Build and deployment** configure:

```
Source:
Deploy from a branch
```

6. Seleccione:

```
Branch:
main

Folder:
/ (root)
```

7. Presione **Save**.

Después de unos minutos el proyecto estará disponible en:

```
https://USUARIO.github.io/NOMBRE-REPOSITORIO/
```
